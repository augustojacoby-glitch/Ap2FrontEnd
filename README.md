# 🪓 Projeto Website - Grêmio Foot-Ball Porto Alegrense

Este é um projeto acadêmico de um website responsivo dedicado ao Imortal Tricolor, desenvolvido como parte dos requisitos de avaliação da disciplina. O objetivo principal foi aplicar conceitos de desenvolvimento web estruturado com HTML5 e o framework **Bootstrap 5**.

---

## 📄 Páginas do Projeto

O projeto é composto por **três páginas HTML** totalmente interligadas através de um menu de navegação funcional, cumprindo o critério de múltiplas páginas:

1. **`index.html` (Início):** Página principal que serve como a "Home" do clube, apresentando um carrosel de imagens, destaques do elenco atual, ídolos históricos e o formulário para sócios.
2. **`sobre.html` (Sobre):** Página dedicada à história do Grêmio, grandes conquistas mundiais e continentais, dados estatísticos e curiosidades.
3. **`contato.html` (Contato):** Área oficial de "Fale Conosco", contendo as informações úteis de localização da Arena do Grêmio e e-mail de suporte para os adeptos.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica de todo o conteúdo.
- **Bootstrap 5 (CDN):** Utilizado para estilização rápida, responsividade e componentes interativos.
- **Flexbox Layout:** Utilizado de forma explícita em seções estratégicas (como no alinhamento do formulário de associação) para garantir uma distribuição flexível e moderna dos elementos.

---

## 🧩 Componentes do Bootstrap Implementados

Para atender às exigências práticas, foram integrados diversos componentes nativos do framework:
- **Navbar (Barra de Navegação):** Menu responsivo com efeito *dropdown* para a listagem de jogadores.
- **Carousel (Carrossel):** Slides de imagens na página inicial com transições automáticas.
- **Cards (Cartões):** Organização visual dos perfis dos jogadores atuais e das lendas históricas.
- **Modal (Janelas Flutuantes):** Modais interativos acionados por botões para exibir a história detalhada de ídolos como Luis Suárez, Renato Gaúcho e Danrlei.
- **Collapse (Efeito Recolhível):** Utilizado para exibir curiosidades ocultas e otimizar o espaço do ecrã.
- **Forms (Formulários):** Formulário completo de candidatura para novos sócios com validações de estilo do Bootstrap.
